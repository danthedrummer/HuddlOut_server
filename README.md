## API Documentation
### Authentication
####Check user authentication

**GET:** *api/auth/checkAuth*

**PARAMS:** *authKey*

**EXAMPLE:** *api/auth/checkAuth?authKey=12345*

**RETURNS:** *int result (0 = invalid token, 1 = requires login, token = successful)*
